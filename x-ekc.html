Empirical verification of Environmental Kuznets Curve

Tomasz Przechlewski

June/2024

Problem

Environmental Kuznets Curve (EKC) hypothesis assumes that relation between CO2 emission per capita (E) and GDP per capita is of inverted U type:

\[E = a + bGDP + cGDP\]

where (\(c <0\))

The data

The data comes from the WorldBank databse. The following indicators were used:

  • EN.GHG.CO2.MT.CE.AR5 (co2) = emission in tons per/capita

  • NY.GDP.PCAP.KD (gdp) = GDP per capita (constant 2015 US$)

  • NY.GDP.PCAP.PP.KD (gdppp) = GDP per capita, PPP (constant 2017 international $)

The time interval analysed is 19902020

wdi.indicators <- c(
  'EN.GHG.CO2.MT.CE.AR5', ## CO2 emission in tons per/capita
  'NY.GDP.PCAP.KD',    ## GDP per capita (constant 2015 US$)
   ##'NY.GDP.PCAP.CD', #= GDP per capita (current US$) 
  'NY.GDP.PCAP.PP.KD'  ## GDP per capita, PPP (constant 2017 international $)
                       ## PPP= purchase power parity
)
year.start <- 1990

f0 <- WDI(wdi.indicators, country = "all", start=year.start)

## Select appropriate columns.
## Change colum names to something less complicated
f1 <- f0 %>% 
  ## Select relevant variables
  ## name1 = name2 denotes _rename_ name2 as name1
  ## eq. EN.ATM.CO2E.PC is renamed to co2 (which is simpler)
  select (
          country, 
          iso3c, 
          year, 
          co2 = EN.GHG.CO2.MT.CE.AR5, 
          gdppp = NY.GDP.PCAP.PP.KD, 
          gdp = NY.GDP.PCAP.KD ) %>%
  ## Compute variables
  mutate(
      lco2 = log(co2),
      lgdp = log(gdp),
      lgdp2 = log(gdp)^2,
      lgdppp = log(gdppp),
      lgdppp2 = lgdppp^2 
  )
